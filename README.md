> An Interactive Website Interface for Endangered Wildlife Awareness and Conservation Action in Malaysia

## 📄 About

WildGuard Malaysia is a website interface designed to raise public awareness about Malaysia's endangered wildlife species. It features a species profile directory, conservation news, and a community engagement portal linking users to volunteer and donation opportunities.

## 🌐 Pages

| Page | Description |
|------|-------------|
| `index.html` | Homepage with hero, featured species & stats |
| `species.html` | Species directory with filter & search |
| `news.html` | Conservation news with sidebar |
| `get-involved.html` | Pledge form, volunteer & donation links |
| `about.html` | Project info, SDG alignment & timeline |

## 🚀 Deploying to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → **Add New Project**
3. Import your GitHub repository
4. Framework Preset: **Other** (static site)
5. Click **Deploy** — done! ✅
